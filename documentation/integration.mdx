# Razorpay Integration

# API Operations Overview

## Orders
1. **Create order** — Create a new order.  
2. **Fetch all orders** — Fetch orders.  
3. **Fetch order details by ID** — Fetch order details using the order ID.  
4. **Fetch all payments related to an order** — Fetch payment details associated with an order.  

---

## Dispute
1. **Fetch all disputes** — Fetch all disputes raised by customers.  
2. **Fetch dispute details by ID** — Fetch dispute details using the dispute ID.  

---

## Invoices
1. **Fetch all invoices** — Fetch details of all invoices.  
2. **Fetch invoice details by ID** — Fetch invoice details using the invoice ID.  
3. **Fetch all invoices of a subscription** — Fetch details of all invoices using the subscription ID.  

---

## Payments
1. **Fetch all payments** — Fetch payments.  
2. **Fetch payment details by ID** — Fetch payment details using the payment ID.  
3. **Fetch card details for a payment** — Fetch card details for a payment.  

---

## Payment Link
1. **Create payment link** — Create a payment link using basic details such as amount and expiry date.  
2. **Fetch all payment links** — Fetch details of all standard payment links.  
3. **Fetch payment link details by ID** — Fetch payment link details using the payment link ID.  

---

## Refunds
1. **Fetch all refunds** — Fetch details of all refunds.  
2. **Fetch refund details by ID** — Fetch refund details using the refund ID.  

---

## Settlements
1. **Fetch all settlements** — Fetch details of all settlements.  
2. **Fetch settlement details by ID** — Fetch settlement details using the settlement ID.
